BIOS

Библиотека BIOS

Что это такое?

BIOS - это набор микропрограмм, реализующий взаимодействие с аппаратурой компьютера и устройствами. Относится к системному программному обеспечению. В нашем случае BIOS непосредственно задает базовую частоту процессора, а так же частоты и тайминги для памяти, обеспечивая их взаимодействие между собой. Так же в нем запрограммированы напряжения системы питания.

В каких случаях менять?

BIOS меняется в целях изменения производительности. Без вмешательства извне, повреждения биоса - очень редка поломка. Установка некорректного или неверно модифицированного биоса может привести к поломке или полному отказу видекарты. Менять на свой страх и риск!

Процесс замены

Подготовка

Важно! Если видеокарта двухпроцессорная, то не стоит перепрошивать её подобным способом!

Узнайте модель и другие характеристики видеокарты, а также текущую версию ее BIOS. С этой целью удобно использовать утилиту GPU-Z, которую можно скачать здесь.

Необходимо узнать следующие данные

  • название модели видеокарты (Name)
  • текущую версию BIOS (BIOS Version)
  • производителя видеокарты (Subvendor)
  • идентификатор устройства (Device ID)
  • тип видеопамяти (Memory Type)
  • размер видеопамяти (Memory Size)
  • ширину шины видеопамяти (Bus Width)
  • частоты графического ядра (GPU Clock) и видеопамяти (Memory Clock)

Сделайте резервную копию текущей версии BIOS. В дальнейшем ее можно будет использовать для возврата видеокарты в прежнее состояние, если процесс перепрошивки закончится неудачно. Нажимаем лкм по указанной иконке, в меню нажимаем на "Save to file". Лучше сохранить с запоминающимся названием, пример - oldbios.rom.

Теперь устанавливаем новую версию BIOS. Подбираем биос согласно характеристикам имеющейся видеокарты.

Полная библиотека BIOS доступна здесь

В результате загрузки Вы должны получить файл прошивки с расширением .rom.

Далее скачиваем утилиту для перепрошивки.

Для видеокарт NVidia (Geforce) нужна утилита NVFlash. Состоит она из двух файлов: NVFLASH.exe и CWSDPMI.EXE. Чтобы скачать архив с указанными файлами нажмите сюда.

Для видеокарт ATI/AMD (Radeon) требуется утилита ATIFlash. Она представляет собой один файл atiflash.exe. Чтобы скачать архив с этим файлом нажмите сюда.

Теперь нужно создать загрузочную флэшку с DOS. Для этой цели подойдет любая флешка, но все находящиеся на ней данные будут уничтожены. После перепрошивки флешку можно будет отформатировать и использовать как раньше. Процесс создания загрузочной флешки с DOS подробно описан здесь.

Далее копируем на флешку следующие файлы

  • старую версию BIOS
  • новую версию BIOS или откорректированную старую BIOS
  • файлы нужной утилиты для прошивки BIOS

Перепрошивка

Сперва запускаем компьютер через флешку с утилитой. Для этого во время запуска компьютера включаем Boot menu, во время запуска будет показана кнопка отвечающая за это. Выбираем флешку. На экране должно появиться несколько строк как показано ниже.

Во многих случаях для видеокарт GeForce требуется отключить защиту от изменения BIOS. С этой целью после приглашения "С:\>" необходимо ввести команду nvflash -protectoff , нажать клавишу Enter и дождаться появления приглашение "С:\>". Для видеокарт Radeon отключать защиту обычно не требуется и этот этап можно пропустить.

Для перепрошивки старого BIOS, после приглашения "С:\>" необходимо ввести одну из следующих команд

  • для видеокарт GeForce: nvflash -4 -5 -6 newbios.rom
  • для видеокарт Radeon: atiflash –p –f 0 newbios.rom

После ввода команды нужно нажать клавишу Enter и немного подождать, пока опять не появится приглашение "С:\>". Его появление свидетельствует о завершении процесса перепрошивки.

После перепрошивки перезагрузить компьютер в обычном режиме. Обычно после перепрошивки требуется переустановить драйвер видеокарты. Драйвера можно найти на оффициальных сайтах NVidia и AMD.
